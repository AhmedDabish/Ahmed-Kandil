## ✨ Features

- 🌙 **Dark / Light Mode** — مع 5 ثيمات لونية مختلفة
- 🌍 **Bilingual** — عربي / English مع تبديل فوري
- 🖱️ **Custom Cursor** — ضرس متحرك مع glow effect
- 🦷 **Orbiting Tooth** — أيقونة ضرس تلف حول الصورة الشخصية
- ⌨️ **Typing Effect** — نص يُكتب ويُمحى تلقائياً
- 📊 **Animated Stats** — دوائر تقدم متحركة
- 🖼️ **Portfolio Gallery** — معرض أعمال مع Modal & Lightbox
- 📱 **Fully Responsive** — متجاوب مع جميع الشاشات
- 🎨 **Theme Presets** — Default / Blue / Royal / Midnight / Emerald
- 📈 **Scroll Progress Bar** — شريط تقدم في أعلى الصفحة
- ✉️ **Contact Form** — نموذج تواصل مع Toast notifications
- 🔄 **Reveal Animations** — عناصر تظهر عند التمرير

